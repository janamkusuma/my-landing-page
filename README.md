# My Landing Page

This is a **React Landing Page** project with the following features:

- Hero section with a consultation form  
- Projects section (displays 3 projects with images)  
- Happy Clients section (displays 3 clients with images)  
- Contact Form section (submits user details)  
- Newsletter subscription section (centered and functional)  
- Admin Panel (to manage Projects, Clients, Contacts, Subscribers)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/janamkusuma/my-landing-page.git
cd my-landing-page
Install dependencies:

npm install


Start the development server:

npm start


Open http://localhost:3000
 to view in the browser.
 Backend APIs (Optional)

The project uses APIs for:

/api/projects → fetch/add/delete projects

/api/clients → fetch/add/delete clients

/api/contacts → fetch/add/delete contact forms

/api/subscribers → fetch/add/delete subscribers

Project Structure
src/
  assets/        # Images (Hero, Projects, Clients)
  components/    # Projects, Clients, ContactForm, Newsletter
  pages/         # LandingPage, AdminPanel
  App.js         # Router and main app

Technologies Used

React.js

Tailwind CSS

Axios (for API calls)

React Router DOM
